# 文本文件读取节点

**DynamicLoadTextFileNode**

- 可以通过传入的路径读取硬盘上任意的文本文件
- 可以选择读取编码 (默认是 `utf_8`)
- 通过 exception 输出端可以获取读取文件时发生的异常 (如果有)


## 参数

**输入**

|  参数名   |   说明   |      注释      |
| :-------: | :------: | :------------: |
| file_path | 文件路径 |                |
| encoding  | 文件编码 | 默认是 `utf_8` |


**输出**

|  参数名   |     说明     |                         注释                          |
| :-------: | :----------: | :---------------------------------------------------: |
|  content  |   内容输出   | 若文件读取成功返回文件内容 (字符串), 否则返回 `None`  |
| exception | 异常信息输出 | 若读取文件过程出现异常则返回异常对象, 否则返回 `None` |
